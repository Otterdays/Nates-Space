**AI Agent Swarm Architecture (The Neo-Label Engine)**  
Dashboard core: Artist seeds (track snippet, vibe, persona, target demo). 5 autonomous agents run 24/7 in parallel, self-optimizing via prompt chaining. Built on Grok/Claude/Midjourney/Runway stack + scheduling API.  
- **Virality Scout Agent**: Scrapes trending sounds/challenges (TikTok/IG/X/YouTube), scores seed track fit (genre-mash potential, hook length). Outputs daily “strike windows.”  
- **Witty Content Agent**: Generates platform-native posts — IG captions, X threads, TikTok scripts, witty hooks. Style: meme-literate, relatable, zero cringe. Auto-A/B tests variants.  
- **Visual Prompt Vault Agent**: Crafts hyper-optimized image-gen prompts (Midjourney/SD/Flux) for album art, meme templates, promo visuals, thumbnail packs. Includes negative prompts + style references. Outputs ready-to-run prompts + variants.  
- **Engagement/Remix Agent**: Simulates replies, drafts UGC prompts for fans (“make your own dance”), suggests remixes/collab angles. Tracks sentiment velocity.  
- **Meta-Prompt Engineer Agent**: Self-improves all other agents’ prompts in real-time (chain-of-thought + few-shot from historical wins). Stores “prompt bible” of proven templates (e.g., “YeeHaw Challenge” template).

**Content & Prompt Output Types (Agent Deliverables)**  
- Awesome/Witty Posts: 30 variants/day per platform — caption + emoji + hashtag + call-to-action (e.g., “Drop your cowboy transformation 👇 #OldTownRoad2.0”).  
- Image-Gen Guides: Locked & loaded prompts (e.g., “hyper-detailed cyber-cowboy meme template, 19-year-old artist face, Old Town Road aesthetic, vibrant, viral TikTok style --ar 9:16 --v 6”).  
- Agent Prompt Guidance Vault: Ready meta-prompts (e.g., “You are Virality Scout v3. Optimized for 2026 algo. Analyze this 15-sec clip against top 10 TikTok trends. Score 1-10 + exact challenge format to copy. Output JSON.”).  
- Full Campaign Packs: 7-day posting calendar + UGC starter kit + remix brief.
